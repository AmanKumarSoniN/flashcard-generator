# Flashcard Generator

A responsive Flashcard Generator web application built using React, Redux Toolkit, Formik, Yup, and Tailwind CSS.

## Features

- Create flashcard groups
- Add multiple terms and definitions
- Upload images
- View flashcards
- Delete flashcards
- Download as PDF
- Responsive design
- Data persistence using LocalStorage

## Tech Stack

- React
- Redux Toolkit
- React Router DOM
- Formik
- Yup
- Tailwind CSS
- Vite
